# Track editor for MRSIM

## Made by Wesley 

### Reimporting is now supported

### Support for adding gates, rotation, stacking 5x5s, and snapping padded poles to 5x5 gates
### Gate order and timing is not supported yet, waiting on passage sensing




## To Run
cd into the directory and then
python -m http.server 8000
and then go to http://localhost:8000
